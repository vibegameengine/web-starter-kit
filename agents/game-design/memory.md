# Game Design Memory

- The canonical game-design domain playbook is
  [KNOWLEDGE_CORE.md](./KNOWLEDGE_CORE.md).
- The evidence register is [SOURCES.md](./SOURCES.md); every external principle
  must retain its original population, genre and method limits.
- Game-specific facts belong to task inputs and project documents, never to
  role memory.
- A design claim is complete only when it connects a rule to expected dynamics,
  observable player behavior, a failure alternative and a test.
